ショートコードを利用してブログカードを表示するプラグイン

== Description ==

以下のショートコードを入力することで簡単にブログカードを表示することができます。

`[blog-card href=""]`

リンクさせたいページのURLを`href`に設定してください。
エディターのblogCardアイコンから追加モーダルを利用して簡単に上記ショートコードを挿入することもできます。
非同期でブログカードが生成されるのでページの表示速度を損なうことは最小限に設計されています。

Attention
ブログカード生成時に取得したリンクサイトのデータは1日間保持されます。

== Installation ==

1. Install this plugin.
1. Set link using short code.
